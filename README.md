# MockMind

### AI-Powered Mock Interview Platform

MockMind is an AI-powered mock interview platform designed to help students, freshers, and professionals prepare for technical and HR interviews through realistic interview simulations.

The platform generates personalized interview questions based on the user's selected domain and resume, conducts interviews using webcam and microphone support, evaluates responses using AI, and provides detailed performance reports with improvement suggestions.

## 🏗️ Workflow

1. User registers and logs into the platform.
2. User uploads a resume (optional).
3. User selects interview domain and difficulty level.
4. AI generates personalized interview questions.
5. User grants webcam and microphone permissions.
6. Interview starts with questions presented one at a time.
7. User answers using voice or text.
8. Speech is converted into text in real time.
9. Entire interview session is recorded.
10. AI analyzes responses and generates a detailed report.
11. Reports, scores, and recordings are saved to the user's dashboard.
12. User can review past interviews and track improvement over time.
